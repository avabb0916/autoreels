# Markdown 脚本格式指南

## 基础格式

每个场景使用 `---` 分隔，包含以下元素：

```markdown
---
## 场景标题
场景副标题
**时长**: 10秒
---
```

## 支持的配置选项

### 必填字段

- `## 标题`：场景的主标题（使用 `##` 开头）
- 副标题：标题下方的第一行非配置文本
- `**时长**`：场景持续时间（秒）

### 可选字段

- `**动画**`：动画类型
  - `fade`：淡入淡出（默认）
  - `slide`：从下方滑入
  - `zoom`：缩放效果

- `**标题颜色**`：标题文字颜色
  - 支持 CSS 颜色值：`#FFFFFF`、`rgb(255,255,255)`、`red` 等

- `**副标题颜色**`：副标题文字颜色
  - 支持 CSS 颜色值

- `**背景**`：场景背景色（未来功能）
  - 支持 CSS 颜色值或渐变

## 完整示例

```markdown
# 我的视频脚本

> 这是一个完整的视频脚本示例

---

## 开场
欢迎来到我的频道
**时长**: 5秒
**动画**: zoom

---

## 主要内容
这里是核心信息
**时长**: 10秒
**动画**: slide
**标题颜色**: #FFD700
**副标题颜色**: #FFA500

---

## 结尾
感谢观看！
**时长**: 5秒
**动画**: fade
**标题颜色**: #00FF00

---
```

## 注意事项

1. **场景分隔**：每个场景必须用 `---` 分隔
2. **标题格式**：标题必须使用 `##` 开头（两个井号）
3. **配置格式**：配置项使用 `**键**:值` 格式
4. **时长单位**：时长自动提取数字，可以写 `10秒` 或 `10`
5. **注释**：使用 `>` 开头的行会被忽略
6. **空行**：空行会被自动忽略

## 中英文支持

配置项支持中英文：

- `**时长**` 或 `**Duration**`
- `**动画**` 或 `**Animation**`
- `**标题颜色**` 或 `**Title Color**`
- `**副标题颜色**` 或 `**Subtitle Color**`
- `**背景**` 或 `**Background**`

## 转换为 JSON

使用转换脚本将 Markdown 转换为 JSON：

```bash
node scripts/convert-script.js input/your-script.md
```

输出文件：`public/scenes.json`

## 常见问题

### Q: 副标题可以有多行吗？
A: 目前只支持单行副标题，第一行非配置文本会被识别为副标题。

### Q: 可以不设置动画吗？
A: 可以，默认使用 `fade` 动画。

### Q: 颜色值必须是十六进制吗？
A: 不是，支持所有 CSS 颜色值格式。

### Q: 如何预览效果？
A: 转换后运行 `npm start` 启动 Remotion Studio 预览。
